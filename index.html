Safe URL Injection Practice Lab

Injected (Sanitized) URL Parameter

This page safely reads the URL parameter ?name= and displays it as plain text (escaped), meaning scripts will NOT run.

Sanitized Output:

No parameter detected.

Try it Yourself

Modify your URL like:

https://yourwebsite.com/safe-url-injection-lab.html?name=<script>alert('XSS')</script>

It will be shown as text instead of executing.

How This Works

The page escapes dangerous characters before displaying them:

<  becomes  &lt;
>  becomes  &gt;
"  becomes  &quot;
'  becomes  '

This prevents:

  • Reflected XSS
  • Script execution
  • HTML injection

This does NOT run anything injected into the URL.(sorry)
